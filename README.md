# Hyperspectral Graphs: Reforming the Pure Graph Paradigm for MTS Forecasting in Frequency Domain
This repository is a successor of the previously released repository about FourierGNN which can be found [here](https://github.com/omkaranu04/CNT-Spring26-FourierGNN)

### Environment Setup
You can setup either virtual environment or conda environment, and then install the required packages using the following command
` 
pip install -r requirements.txt
`

### Data Preparation
Follow the steps mentioned in the `Data Preparation` section of the [FourierGNN repository](https://github.com/omkaranu04/CNT-Spring26-FourierGNN)

### Experiments
To run the experiments, you can use the followwing commands
`
python run.py
`
and
`
python run_pred_length.py
`
